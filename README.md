## About
The code is a basic RPG featuring two characters (with the potential for more creatures and playable characters in the future, and even a graphical interface).
## Characters: 
### 1. Violet
A young warrior determined to face the world’s challenges. With a sharp sword and an indomitable spirit, she is ready to take on any opponent who crosses her path.

### 2. Colossus 
An imposing stone giant, with overwhelming strength and almost impenetrable resilience. His eyes gleam with a sinister light, and his presence is as intimidating as the mountain itself. He is a formidable opponent, capable of crushing everything in his path with a single blow.

## Requirements for running the code
The user will need:
- A Lua compiler installed on their computer and added to the operating system’s PATH environment variable
  It can be downloaded from the link🔗 [Download Lua](https://www.lua.org/download.html) 🌙
- Visual Studio, CMD, PowerShell or any other command prompt.
- Creativity, in case you wish to further develop the code; it will be available for copying and collaboration.

## How to run
In the terminal, you only need the Lua extension and the name of the main file.
For example, as I used it:
```
lua55 main.lua
```
